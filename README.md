# business-dashboard

Rserveを利用している飲食店向けのダッシュボード

## 技術スタック

- Vite
- React
- TypeScript
- Ant Design

## 開発環境構築

1. リポジトリをクローン
2. [FastAPIサーバー](https://github.com/Rserve-inc/backend) を起動
3. `npm install`で依存関係をインストール
4. `npm run dev`で開発サーバーを起動
